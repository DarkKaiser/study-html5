Restricting yourself to a namespace

Provides a way of namespacing a socket.

Connect to '/chat' namespace

Please input the Socket.io server to connect.

Codes

Node.js Server with socket.io

var chat = io
		.of('/chat')
		.on('connection', function (socket) {
			// do something
		});

Vert.x Serer with socket.io module for vert.x

final Namespace chat = io.of("/chat");
chat.onConnection(new Handler<SocketIOSocket>() {
	public void handle(SocketIOSocket socket) {
		// do something
	}
});

Socket.io client

var chat = io.connect('http://localhost/chat');

chat.on('connect', function () {
	// do something
});
